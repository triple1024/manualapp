＜データ追加＞

ログインボタンをクリックして
ログインし
djangoの管理画面から行って下さい。

・新しい作業を追加したい時

EDIT MANUAL→作業（追加）を選び→作業名を入力し、保存を押す。

＊編集の場合
EDIT MANUAL→作業（変更）を選び→作業名を入力し、保存を押す。

・新しい手順を追加したい時

EDIT MANUAL→手順（追加）を選び→手順名、関連作業、前と後の手順（無ければ空欄でも可）、を入力し保存を押す。

＊編集の場合
EDIT MANUAL→手順（変更）を選び→、関連作業、前と後の手順（無ければ空欄でも可）、を編集し保存を押す。

・新しいマニュアルを追加したい時

EDIT MANUAL→マニュアル（追加）を選び→手順、内容、関連マニュアル（無ければ空欄でも可）、を入力し保存を押す。

＊編集の場合
EDIT MANUAL→マニュアル（変更）を選び→、手順、内容、関連マニュアル（無ければ空欄でも可）、を編集し保存を押す。
もしくは
EDIT MANUAL→ドキュメント（変更）を選び→、手順、内容、関連マニュアル（無ければ空欄でも可）、を編集し保存を押す。
